E.2. Contributing to This Document

If you have problems or suggestions regarding this document, please mail them to <ubuntu-users@lists.ubuntu.com>.

Please do not contact the authors of this document directly. There is also a discussion list for debian-installer, which includes discussions of this manual. The mailing list is <debian-boot@lists.debian.org>. Instructions for subscribing to this list can be found at the Debian Mailing List Subscription page; or you can browse the Debian Mailing List Archives online. Please do not contact debian-boot about issues specific to Ubuntu.
